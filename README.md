# ⚡ Green Ammonia Investment Valuation PRO
### Strategic Investment Timing Model using Real Options & LSM

This interactive SaaS tool evaluates the economic feasibility and optimal investment timing for **Green Ammonia** projects. It bridges the gap between static DCF models and dynamic market realities.

## 🌟 Key Features
* **Real Options Analysis**: Leverages the Least Squares Monte Carlo (LSM) algorithm to value the "option to wait."
* **Technological Learning Curves**: Models CAPEX reduction (default -8% p.a.) to simulate the benefit of future cost-downs.
* **Stochastic Modeling**: Uses Geometric Brownian Motion (GBM) to simulate 5,000+ paths for Ammonia and Carbon prices.
* **Interactive Dashboard**: Real-time sensitivity analysis for Ammonia prices, capacities, and investment scales.

## 🛠️ Tech Stack
* **Core Logic**: Python (NumPy, Pandas)
* **Interface**: Streamlit
* **Deployment**: Streamlit Community Cloud

## 📈 Business Use Case
Designed for **Renewable Energy Developers**, **Infrastructure PE Funds**, and **Energy Consultants** to determine if they should "Invest Now" or "Wait and See" based on market volatility and tech progress.
